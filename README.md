# Food_List_Combined_Clean
Currently we archive separate years of Fair Shares data into separate Google Sheet documents. This folder was created to create a clean data source that combines multiple archive seasons into one source file. This will allow for easily handling messy data in a single project. As well as easier handling of different season data sets. 

## Requirements
1. Create easily runnable code to pull in each week's new archive data
2. Combine all seasonal files into one source file (overwritten every time via this code)
3. Must add in a new column season column in the output file
4. Clean up any missing records or columns that have missing values (where applicable)
5. Add in additonal cleansed columns (where applicable)
6. Creat a single source of truth that updates so future projects do not have to find multiple seasonal files and contain extra work to clean data
